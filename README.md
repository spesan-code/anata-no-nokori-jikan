# あなたの残り時間

このアプリは、あなたの人生や目標の「残り時間」を可視化し、1分1秒を大切に生きるための匿名Webアプリです。

## ローカルでの実行方法

**前提:** Node.js

1. 依存パッケージのインストール:
   `npm install`
2. アプリの起動:
   `npm run dev`

# ページ構成
- pages/About.tsx: アプリについて
- pages/Terms.tsx: 利用規約
- pages/Contact.tsx: お問い合わせ

# アプリ名
このアプリの正式名称は「あなたの残り時間」です。
